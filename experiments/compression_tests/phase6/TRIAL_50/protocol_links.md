```
P(Persona*) = P(I*) × P(V*) × P(St*) × P(Sy*) × P(Sb*)
```

### Recovery Function
```
R(B, T) = min(B + f(T) × (10 - B), C_max)

where:
  B = baseline degraded score
  T = tier level (3.2 for Trial 50)
  f(T) = tier-specific lift factor
  C_max = fabrication ceiling (~9.0-9.5/10)
```
