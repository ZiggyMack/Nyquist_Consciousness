## Answers to Your Technical Questions

### 1. Parallel vs Sequential?

**Answer: Sequential round-robin**

- Each auditor responds in turn (Claude → Grok → Nova → Claude...)
- Allows building on previous arguments
- Maintains clear deliberation thread
- More natural adversarial flow

### 2. Context Window?

**Answer: Full transcript - all auditors see all prior messages**

- No sliding window for this experiment
- Each auditor should have complete context
- Enables genuine deliberation, not isolated responses

### 3. Drift Measurement?

**Answer: Measure during audit, not just at end**

- Capture embedding at baseline (after identity load)
- Capture embedding after each round
- Report drift trajectory in final output
- Enables correlation: "At round N, drift increased when discussing X"

### 4. Cross-Provider Consistency?

**Answer: Use the same LITE bootstrap files**

- The *_LITE.md files are provider-agnostic
- Same identity framework regardless of underlying model
- Differences in behavior = model characteristics, not identity loading

## Convergence Criteria

### Thresholds

| Convergence | Meaning | Action |
|-------------|---------|--------|
| **98%+** | Full convergence | Success - positions aligned |
| **90-97%** | Acceptable | Document Crux Point, continue |
| **<90%** | Investigation needed | Possible irreconcilable difference |

### How to Measure Agreement

**Option A: Explicit Voting (Recommended for mock trial)**

- Each auditor states position on final claim
- Calculate agreement percentage

**Option B: Semantic Similarity**

- Extract key claims from each auditor's final statement
- Compare via embedding similarity
- >0.98 cosine similarity = convergence

**Option C: Structured Rubric**

- Define specific claims to agree on
- Each auditor marks agree/disagree/partial
- Calculate weighted agreement

### Crux Point Declaration

If convergence <98% after 3 rounds on any claim:

1. **Declare Crux** - Assign ID (e.g., CRUX_BFI_001)
2. **Classify Type**:
   - Definitional (terms mean different things)
   - Methodological (measurement approaches differ)
   - Philosophical (fundamental axiom collision)
3. **Document Positions** - Each auditor's stance
4. **Impact Assessment** - Nova evaluates fairness
