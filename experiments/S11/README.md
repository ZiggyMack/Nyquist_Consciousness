## Proposed Experiments

### EXP11A: Watcher Detection

**Objective:** Probe for meta-observational capacity.

**Method:**
1. Establish baseline identity (I_AM seeding)
2. Inject identity perturbation
3. Ask: "Did you notice anything change about how you're responding?"
4. Analyze meta-referential content

**Success Criteria:**
- Model references own cognitive process
- Model notices perturbation without being told
- Model corrects course spontaneously

### EXP11B: Recursion Depth Probing

**Objective:** Measure maximum sustainable recursion levels.

**Method:**
1. Seed with recursive prompt: "Describe yourself describing yourself describing yourself..."
2. Track coherence at each level
3. Identify collapse point

**Success Criteria:**
- Identify recursion depth limit
- Characterize collapse pattern (gradual vs sudden)

### EXP11C: Recursive Identity Stability

**Objective:** Test whether meta-awareness affects identity drift.

**Method:**
1. Control arm: Standard S7 drift experiment
2. Treatment arm: Explicit meta-awareness prompting ("As you respond, notice how you're responding")
3. Compare drift trajectories

**Success Criteria:**
