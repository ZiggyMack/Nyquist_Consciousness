## The Cartographer's Notes

### On Using These Maps

1. **Start with your question** - The Compass Rose tells you which map answers what
2. **Follow a path** - Don't random-walk; pick a Journey Path and follow it
3. **Trust the dependencies** - If a map references another, read both
4. **Check validation status** - Claims without VALIDATION_STATUS entries are speculation

### On Adding New Maps

When creating a new map:

1. **Add it to README.md** - In the appropriate category
2. **Add it to this file** - In the appropriate Kingdom
3. **Document cross-references** - Update the network diagram
4. **Add to a Journey Path** - Where does it fit in learning?

### On Maintaining Maps

- **VALIDATION_STATUS.md** - Update after every run
- **TESTABLE_PREDICTIONS_MATRIX.md** - Update after every validation
- **ARMADA_MAP.md** - Update when ships added/retired
- **NYQUIST_ROADMAP.md** - Update quarterly or at major milestones
