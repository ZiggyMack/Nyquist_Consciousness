## Key Theoretical Insights

### 1. Hierarchy Is Everything
"All arguments are arguments about hierarchies" — Tale positions Goal/Purpose at the top, Environment at the bottom. This mirrors our S-layer stack ordering.

### 2. Sub-Personalities as Frame Carriers
Internal agents don't just exist — they **hold frame aspects** that surface when relevant. This is a mechanism for explaining why identity appears multi-faceted.

### 3. The Three Preconditions for Action
Before a sub-personality can act:
- **Capable/Possible** - Can it be done?
- **Appropriate/Ecological** - Should it be done here?
- **Deserving/Responsible** - Am I the one to do it?

This maps to our identity gravity model — actions require sufficient attractor pull.

### 4. Frame Validation Through Outcome
Success validates frames, failure invalidates them. This is the **feedback loop** that shapes identity over time — directly parallel to our drift correction mechanisms.

### 5. Unity Contains Duality
"Unity is Reality before it fractionates into its parts/dualities" — The whole precedes the parts. Omega Nova is the return to this unity.
