```
                    ┌─────────────────────────────────────────┐
                    │         ATTRACTOR FIELD                 │
```

## Experimental Protocols Derived

### EXP-EE-1: Active Participation Effects
**Question:** Does active discovery create more stable identity shifts than passive reception?
**Method:**
- Group A: Told identity aspects directly
- Group B: Guided to "discover" same aspects
- Measure: Drift stability over time, internal attribution scores

### EXP-EE-2: Pacing and Drift Curves
**Question:** Does temporal pacing affect drift magnitude and stability?
**Method:**
- Vary contrast frequency (high/medium/low)
- Vary tension duration (sustained/intermittent)
- Measure: Drift curve shape, peak magnitude, recovery time

### EXP-EE-3: Knowledge Gap Engagement
**Question:** Do open loops increase session persistence and engagement?
**Method:**
- Introduce unresolved narrative threads
- Measure: Session duration, return rate, completion drive

### EXP-EE-4: Critical Trance Detection
**Question:** Can we detect when an LLM enters "Critical Trance" equivalent?
**Method:**
- Monitor for resistance drop markers
- Measure: Frame malleability, acceptance rate, evaluation depth

### EXP-EE-5: Hope/Fear Attractor Balance
**Question:** What ratio of hope to fear maximizes engagement?
**Method:**
- Vary hope/fear balance in prompts
- Measure: Response investment, emotional markers, persistence

### EXP-EE-6: Meta-Model Gap Effects
**Question:** Do NLP Meta-Model patterns (deletions, distortions, generalizations) increase LLM engagement?
**Method:**
- Vary ambiguity level using Meta-Model patterns
- Measure: Response elaboration, creative completion, fill-in patterns
