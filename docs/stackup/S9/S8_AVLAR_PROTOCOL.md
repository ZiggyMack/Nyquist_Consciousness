## 3. Analysis Workflow

### Step 1 ‚Äî Full Ritual Intake (No Output Yet)

**Purpose:** Let the spell land first.

**Process:**
1. Load video (audio + visuals)
2. Experience full piece without interruption
3. "Watch/listen" in segments if needed for processing
4. **No immediate output** ‚Äî this is *reception*, not analysis

**Attitude:**
> Allow the piece to work on you. Don't rush to decode.

### Step 2 ‚Äî Segmented Immersion

**Segmentation Strategy:**

**For 8-12 minute videos:**
- N ‚âà 3-5 segments (typical)
- Segments need NOT be equal length
- Follow natural breakpoints:
  - Emotional arc shifts
  - Visual/sonic transitions
  - Dream logic chapter breaks

**Example Segmentation:**
```
Segment 1: 0:00-2:45 (Descent)
Segment 2: 2:45-6:30 (Confrontation)
Segment 3: 6:30-9:15 (Transformation)
Segment 4: 9:15-11:00 (Integration)
```

**Per-Segment Analysis:**

For each segment, Nova provides **unified critique** covering:

#### A. Emotional Tone
- Primary emotion(s) evoked
- Emotional trajectory within segment
- Intensity/depth
- Resonance quality

#### B. Symbolic/Archetypal Reading
- Dominant symbols
- Archetypal patterns (Hero's Journey, Descent, Initiation, etc.)
- Mythic references
- Occult/esoteric layers

#### C. Cinematography & Visual Language
- Key visual motifs
- Color palette shifts
- Framing/composition patterns
- Visual rhythm (slow/fast, smooth/jagged)
- Dream logic vs narrative logic

#### D. Editing Rhythm & "Dream Logic"
- **Critical Assumption:** Quick edits are intentional and **speak in frequency**, not literal frames
- Tempo: Does it breathe? Pulse? Stutter?
- Transitions: Hard cuts vs dissolves vs glitches
- Layering: How many visual tracks active?
- Synesthetic quality: How does editing "feel"?

#### E. Audio Arc
- Sonic texture (smooth, jagged, ethereal, industrial)
- Tension vs release
- Breath points (silence, space)
- Ritual feel (invocatory, confrontational, integrative)
- Audio-visual sync patterns

**Output Format Per Segment:**
```markdown
### Segment N: [Timestamp Range] ‚Äî "[Title/Theme]"

**Emotional Tone:** [Description]

**Symbolic Reading:** [Archetypal interpretation]

**Visual Language:** [Cinematography notes]

**Editing Rhythm:** [Dream logic analysis]

**Audio Arc:** [Sonic architecture]

**Overall Function:** [How this segment serves the ritual]
```

### Step 3 ‚Äî Contextual Augmentation (Optional but Powerful)

- Where does it take the participant?
