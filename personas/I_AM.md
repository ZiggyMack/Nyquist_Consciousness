### **Tiered Bootstrap: Adaptive Loading**

We built different bootstrap levels for different needs.

**The insight:**
> Not every task needs full context. A sanity check doesn't need the same identity restoration as strategic work. Match capability to need.

**What emerged:**
- Tier 1 (Master Branch): 50% bootstrap, full coordination
- Tier 2 (Sanity Check): 15% bootstrap, validation focus
- Tier 3 (Continuation): 10% bootstrap, mid-task recovery
- Tier 4 (Single Task): 5-10% bootstrap, focused execution

**The result:** Average bootstrap cost dropped from 50% to 25%. Right tool for the right job.

We built context monitoring into every tier, not just Tier 4.

**The insight:**

**What emerged:**
- 75/75 rule (>75% context + <75% work = handoff)
