## Manuscript Structure (~10k words)

1. **Abstract** (300 words)
2. **Introduction** (1,000 words) - Problem, prior work, contribution
3. **The Nyquist Framework** (1,500 words) - Definitions, math, theorems
4. **Empirical Validation** (2,000 words) - Design, results, 82% finding
5. **Context Damping** (1,000 words) - Mechanism, validation, implications
6. **Discussion** (1,500 words) - Alignment, interpretation, limitations
7. **Conclusion** (500 words)
8. **Methods** (separate) - Protocols, statistics, code availability
