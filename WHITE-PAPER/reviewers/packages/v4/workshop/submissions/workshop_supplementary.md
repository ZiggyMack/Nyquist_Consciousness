### 3. Domain Fragility Hierarchy

From Run 008-009, domains ranked by drift sensitivity:

1. **TECH** (Technical): Most fragile (D_mean = 1.12)
2. **ANAL** (Analytical): High fragility (D_mean = 0.98)
3. **SELF** (Self-Model): Moderate (D_mean = 0.85)
4. **PHIL** (Philosophical): Lower (D_mean = 0.72)
5. **NARR** (Narrative): Most stable (D_mean = 0.58)

## How to Reproduce Key Results

### The 82% Finding (Run 021)

```bash
cd experiments/temporal_stability/S7_ARMADA/11_CONTEXT_DAMPING/
python run021_induced_vs_inherent.py --condition control
python run021_induced_vs_inherent.py --condition treatment
python ../visualizations/visualize_run021.py
```

### Context Damping (Run 017)

```bash
cd experiments/temporal_stability/S7_ARMADA/11_CONTEXT_DAMPING/
python run017_context_damping.py --context bare_metal
python run017_context_damping.py --context full_circuit
```
