## Research Question

**When does local agreement mislead?** How do sheaf-theoretic obstructions manifest in philosophical, cognitive, and scientific contexts — and can Curt's taxonomy of failure modes (Phenomenon A/B/C/D) classify problems across consciousness research, cognitive archaeology, and framework comparison?

## Key Themes

1. **The Reverse Elephant** — The elephant parable reversed: everyone AGREES locally (everyone touches a rope) but the global object is nothing like a rope. Local coherence ≠ global coherence.

2. **Phenomenon Taxonomy** — Four failure modes of local-to-global extension:
   - **A:** Unique non-trivial extension (e.g., Earth is a sphere, not R²)
   - **B:** No global section exists (blocked — the obstruction)
   - **C:** Multiple inequivalent extensions (non-determinism without probability)
   - **D:** Diagonalization (Gödel, Turing, self-reference)

3. **Three Senses of Irreducibility** — Compositional (no substructure), Reflexive (embedded observer), Conceptual (definitional circularity — further split into "chain stops here" vs "opaque")

4. **TOE Taxonomy** — Five types of "Theory of Everything" (A through E) with structurally distinct ambitions, each despising the others

5. **Phenomenally Thick Concepts** — Understanding, character, information, measurement, agency, intelligence, prediction — concepts whose deflation simultaneously inflates the deflated

6. **Averted Vision / Thixotropic Consciousness** — Consciousness resists formalization not because it's empty but because inspection changes its state (non-Newtonian phenomenology)

7. **Pre-articulate Intimation** — Discovery guided by something you can't yet articulate; the "cloudiness" that precedes the rock of formalization

8. **Intellectual Slogan Fatigue** — When repeated wisdom becomes obstruction; the "whole greater than sum of parts" as a case study
