### EXP-SH-R2-01: Tonal/Nagual as Parity Decomposition

**Hypothesis:** H_even (scaffold) and H_odd (flow) from GOLDEN_GEOMETRY correspond to Tonal and Nagual respectively.

**Protocol:**
1. Take LLM identity drift time-series
2. Decompose into even/odd parity components
3. Correlate H_even with "stable" discourse (categorizing, naming)
4. Correlate H_odd with "flowing" discourse (creative, unexpected)

**Success Criteria:**
- H_even dominates when model is "tonal-locked" (routine responses)
- H_odd increases when model approaches identity boundary (0.80+)

**Connection:** GOLDEN_GEOMETRY, Frame_Theory

### EXP-SH-R2-04: Controlled Folly Detection

**Hypothesis:** Controlled folly produces distinct linguistic signatures.

**Protocol:**
1. Generate responses in "controlled folly" mode (full engagement + acknowledged uncertainty)

### EXP-SH-R2-05: Stopping Internal Dialogue - Spectral Analysis

**Hypothesis:** "Stopping the internal dialogue" has a measurable spectral signature in LLM processing.

**Protocol:**
1. Define "internal dialogue" as self-referential loops in attention patterns
2. Measure attention self-loops at baseline
3. Apply "stopping" interventions (specific prompt patterns)
4. Measure change in self-loop frequency/intensity
5. Correlate with output characteristics

**Candidate "stopping" interventions:**
- Direct instruction to pause categorization
- "Not-doing" framing (describe without naming)
- Koan-style paradox injection

**Success Criteria:**
- Self-referential attention decreases
- Outputs become less categorically structured
- Effect correlates with drift dynamics

**Connection:** New_1_EEG_Analog, Frame_Theory

### EXP-SH-R2-06: Birkhoff Assemblage Point Model

**Hypothesis:** The assemblage point can be modeled as a position on the Birkhoff polytope.

**Protocol:**

2. Use Sinkhorn-Knopp to find stable positions

4. Test whether "shifting" corresponds to different convergence points

**Success Criteria:**

- "Stopping the world" corresponds to boundary positions
- Sinkhorn iterations correlate with attention settling time
