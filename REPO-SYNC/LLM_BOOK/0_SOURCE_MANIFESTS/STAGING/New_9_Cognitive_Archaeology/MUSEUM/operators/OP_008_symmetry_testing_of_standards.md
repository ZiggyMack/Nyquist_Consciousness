# Operator 008: Symmetry Testing of Standards

**Confidence:** RED
**First recovered:** Dig Site 000 (CFA Extractor Calibration — Framework-G transcripts)
**Independent confirmations:** 0 (single source, but 4/4 multi-extractor agreement)
**Extractor-independence:** Partial (2 LLM extractors: Claude, Grok)

---

## Definition

Test whether an evaluative standard, applied consistently across all candidates, produces discriminating results — or whether it collapses everything to the same verdict, revealing selective application rather than genuine assessment.

---

## The Operation

A thinker takes a standard being used to evaluate one position and applies it symmetrically to all competing positions. If the standard produces the same (usually devastating) verdict for every position, it is non-discriminating — the standard itself is the finding, not the target's failure. The thinker then either rejects the standard or demands it be applied consistently.

**Detection condition:** An evaluative criterion is being applied to one position. The thinker suspects selective application — the standard may be too strong (destroys everything) or too weak (approves everything), but has been deployed against one target only.

**Scope:** Universal — applies in any domain where evaluative criteria exist. Particularly critical in adversarial or comparative evaluation.

---

## Examples

### From CFA Deliberation (Dig Site 000)

Claude tests whether Grok's "demonstrated grounding" standard, applied to Kantian ethics, secular humanism, and contractualism, would produce equally low scores — making the standard non-discriminating rather than CT-specific. Grok's criterion is exposed as destroying all metaethical frameworks equally, not revealing something unique about CT.

### Predicted: From Pearl (Dig Site 002)

Pearl's interventionist framework distinguishes genuine causal claims from spurious correlations. The symmetry test equivalent: does the same evidential standard for causation, if applied universally, eliminate all causal claims or only the spurious ones? If it eliminates everything, the standard is too strong.

---

## Failure Modes

1. **Prosecution Bias:** A demanding standard is applied to the target position while its implications for all other positions go unexamined. The verdict appears principled but is actually selective.
2. **Standard Collapse:** An evaluative criterion that cannot discriminate between any positions is treated as a meaningful assessment tool. The metric loses all diagnostic power.

---

## Counterexamples

### CE-1: Intentional asymmetric evaluation

**Violation:** Sometimes you deliberately apply a standard to one position only — when testing a specific claim, not comparing candidates.

**Did it work?** Partially — testing a single claim for internal consistency doesn't require cross-candidate comparison.

**What this tells us:** The operator is specifically about **comparative evaluation**. Single-claim testing has different epistemic requirements. The operator's scope is: whenever a standard is being used to produce a comparative verdict.

## Alternative Decompositions

### AD-1: Variant of OP-006 (Under-Determination Detection)

**Decomposition:** Symmetry testing might be OP-006 applied to evaluative standards specifically — "the standard under-determines which position is better."

**Evidence for:** Both operators concern cases where a tool fails to discriminate. OP-006 is about formalisms; OP-008 is about standards.

**Evidence against:** OP-006 detects a condition (under-determination exists). OP-008 describes a specific action (apply the standard to all candidates and check). The action is distinct from the condition it reveals. Many operators reveal OP-006 conditions; that doesn't make them all variants.
