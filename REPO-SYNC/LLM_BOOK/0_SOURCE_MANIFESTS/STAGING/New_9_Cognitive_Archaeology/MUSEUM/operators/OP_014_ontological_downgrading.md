# Operator 014: Ontological Downgrading

**Confidence:** RED
**First recovered:** Dig Site 002 (Barandes, via Nova synthesis)
**Independent confirmations:** 1
**Extractor-independence:** Unknown
**Class:** Translation Operator

---

## Definition

When something is treated as either fully real or fully unreal, replace the binary with a graded scale — ask not "is X real?" but "what degree of reality does X have, and what kind?"

---

## The Operation

Encountering a claim that X is "real" or "not real," translate the binary into a spectrum:

- Fundamental (particle positions, forces) — maximally real
- Emergent-but-real (temperature, pressure, chairs) — genuinely real, not fundamental
- Perceptual (color, wetness) — real experiences, not intrinsic properties
- Mathematical construct (wave functions in ISP) — useful tools, not physical objects

The operation is a downgrade (or upgrade) — moving an entity to its correct position on the spectrum rather than forcing it into a binary. Crucially: downgrading something from "fundamental" to "emergent" does NOT make it unreal. Emergent things can be real.

**Detection condition:** When a debate is stuck on "is X real?" and both sides seem partially right. This usually means X is real but at a different level than the debaters assume.

**Scope:** Universal — applicable to any domain where reality/unreality is debated. Especially productive in: consciousness debates (is qualia real?), social ontology (are nations real?), ethics (are moral facts real?), emergence (are patterns real?).

---

## Examples

### From Barandes (Dig Site 002)

The wave function debate: is it real or not? Standard answers: (1) it's the most real thing (Many-Worlds), (2) it's just a calculation tool (instrumentalism), (3) we can't know (agnosticism). Barandes' move: configurations are real, probabilities are real, wave functions are mathematical tools that encode those probabilities. The wave function gets ontologically downgraded from "physical entity" to "representation" — but the underlying probabilities get UPGRADED from "mere tools" to "genuine features of reality." Graded realism dissolves the binary.

### Barandes on Griffiths' trichotomy

David Griffiths' Introduction to QM offers three positions: realist (wave function is physically real), orthodox (wave function collapses on measurement), agnostic (we can't know). Barandes says this trichotomy "steamrolls important nuances" — all three assume the wave function is the central object. Ontological Downgrading says: what if the wave function isn't the right level of description? What if the question itself is malformed because it assumes binary ontology?

## Failure Modes

1. **Everything Is Equally Real:** Refusing to discriminate — treating all levels as equivalent. Graded realism requires genuine discrimination between levels. Configurations are MORE real than wave functions in ISP. Temperature is LESS fundamental than molecular motion but still real.
2. **Downgrading as Dismissal:** Using "it's just emergent" as a way to say "it's not real." Emergence ≠ unreality. This is the failure mode the operator is specifically designed to prevent.
3. **Grading Without Criteria:** Placing things on the spectrum without specifying what determines the ordering. Barandes uses: does it have a formalism-independent existence? Does removing the mathematical description change the physics?
