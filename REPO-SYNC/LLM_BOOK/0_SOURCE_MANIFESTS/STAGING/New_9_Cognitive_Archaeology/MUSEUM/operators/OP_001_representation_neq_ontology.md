# Operator 001: Representation ≠ Ontology

**Confidence:** YELLOW
**First recovered:** Dig Site 001 (Barandes)
**Independent confirmations:** 2 (Barandes, CFA/FUT)

---

## Definition

Separate what the formalism says from what actually exists — a mathematical description that is isomorphic to another cannot claim ontological priority over it.

---

## The Operation

When encountering a claim about what is "real" or "fundamental," translate the claim into an equivalent (isomorphic) representation. If the claim does not survive the translation — if it is true in representation A but false in representation B — then the claim is about the representation, not the ontology. The thinker is confusing their map for the territory.

Input: A claim presented as ontological ("branches vanish," "causes are fundamental," "this is the natural way to see it").
Output: A verdict — is this a feature of reality, or a feature of the description?

---

## Examples

### From Barandes (Dig Site 001)

Hilbert space axioms are mathematically isomorphic to a system of classical harmonic oscillators. In the oscillator picture, a zero-amplitude branch corresponds to a spring at rest — still a spring. Therefore the Everettian claim that zero-amplitude branches "vanish" depends on the representation, not the physics. The ontological claim dissolves under translation.

### From CFA / FUT

FUT's qualification procedure: a Framework Under Test (FUT) can be described in its own terms or in the evaluator's terms. If a property exists in one description but not the other, the property belongs to the evaluation framework, not the FUT. This is OP-001 applied to worldview assessment rather than physics.

---

## Failure Modes

1. **Representation Reification:** Treating a mathematically convenient description as physically real. Choosing one isomorphic description over another and claiming ontological privilege.
2. **Translation Refusal:** Refusing to translate into equivalent representations because "this is the standard formalism." Standardness is a sociological fact, not an ontological one.
