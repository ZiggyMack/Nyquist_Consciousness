## Why Dirac?

Dig Site 002's Q50 (Source Discovery — recursion fuel) ranked Dirac as the **#1 candidate** by expected operator yield. Three independent reasons converge:

1. **Inverse cognitive architecture.** Dirac's discovery process appears to be the INVERSE of Barandes'. Where Barandes proceeds philosophically/rigorously (backward-reading from constraints), Dirac proceeds aesthetically/playfully (forward-generating from mathematical beauty). If the Extraction Protocol can handle both, it has real discriminative power.

2. **RCI stress test.** Reverse Constraint Inference (OP-016 candidate) is the strongest pattern from Dig Site 002 — but Dirac's "playful deformity" may represent a discovery architecture that proceeds FORWARD, not backward. If Dirac's process genuinely has no backward-reading phase, RCI is not universal. Either outcome is informative.

3. **Recommended negative control.** PASS Ω Revision 8 recommends the next dig site include a thinker where the protocol should STRUGGLE. Dirac is the best candidate: his aesthetic/intuitive process may resist the structured extraction that worked so well on Barandes' philosophical/systematic process.
