This would mean the Museum needs multiple discovery architecture templates, not one master algorithm. *(Dig Site 002, Unknown #12)*

3. **Are false invariants detectable BEFORE a representation shift, or only after?** Q45 shows Barandes identified false invariants by changing representation. But can the Noether Lens detect them in advance — predict which features WON'T survive before running the experiment? Dirac's approach may shed light: did he PREDICT what was representational before finding the simpler form, or discover it only after? *(Dig Site 002, Unknown #13)*
