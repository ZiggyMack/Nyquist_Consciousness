2. Linearity (State Consistency): To ensure consistency across the wave function substrate, Dirac utilized linear operators. This ensured that regardless of whether the system was expressed via matrices or wave equations, the underlying state logic remained equivalent.
3. Mathematical Beauty (Structural Integrity): For Dirac, "Beauty" was a heuristic for structural stability. Dirac believed the error lay in "unresolved factors" (like spin) rather than the equation’s core logic.

The "So What?" Layer: These constraints allowed Dirac to predict the "positron" through logical necessity. These constraints were used to generate the system’s architecture, moving physics beyond simple ex post facto validation.
