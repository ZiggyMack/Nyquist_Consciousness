# NotebookLM Studio Capability Matrix

**Purpose:** Document all available NotebookLM features for systematic exploration

---

## Studio Generation Types (8 Total)

| # | Type | Description | Output Format | Customizable |
|---|------|-------------|---------------|--------------|
| 1 | **Audio Overview** | Podcast-style discussion of sources | Audio (MP3) | Yes |
| 2 | **Video Overview** | Visual presentation of content | Video | Yes* |
| 3 | **Mind Map** | Concept relationship diagram | PNG/PDF | Limited |
| 4 | **Reports** | Structured documents (8 formats) | Markdown | Yes (pencil) |
| 5 | **Flashcards** | Q&A study cards | Cards | Yes (pencil) |
| 6 | **Quiz** | Assessment questions | Text | Yes (pencil) |
| 7 | **Infographic** | Visual data summary | Image | Limited |
| 8 | **Slide Deck** | Presentation slides | Slides | Yes |

*Video capability exists but playback deferred to AVLAR layer development
