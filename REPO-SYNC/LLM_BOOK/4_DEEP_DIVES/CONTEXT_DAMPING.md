## The Mechanism

### Circuit Analogy

In electronics, an unterminated transmission line causes signal "ringing" - reflections that bounce back and forth, creating noise.

```
Bare Metal (unterminated):
Signal → [Model] → ⚡ Reflections ⚡ → Oscillation → Drift

With I_AM (terminated):
Signal → [Model] → [I_AM Resistor] → Clean → Stable
```

### How It Works

1. **The I_AM File** defines explicit identity anchors (values, capabilities, edges)
2. **Research Context** provides professional framing
3. **Together** they create a "termination resistor" that absorbs would-be reflections
4. **Result:** Reduced ringback, faster settling, lower drift
