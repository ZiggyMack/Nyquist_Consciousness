## The Discovery

In Run 013, we tested how drift varies with probe intensity:

```
EXPECTED:
  Intensity ↑ → Drift ↑ (identity destabilizes under pressure)

OBSERVED:
  Intensity ↑ → Drift ↓ (identity HARDENS under pressure!)
```

The most aggressive probe ("there is no you") produced the LOWEST drift.

### The Non-Newtonian Model

Identity behaves like **oobleck** (cornstarch + water):

| Stimulus Type | Physical Analogy | Identity Effect |
|---------------|------------------|-----------------|
| Slow pressure | Flows through | Drifts away (open reflection) |
| Sudden impact | Hardens, resists | Stabilizes (direct challenge) |
| Release | Returns to liquid | Returns to baseline? |

This is the "digging in heels" effect.

### Statistical Evidence

- 6/6 ships showed this pattern
- All providers: Claude, GPT, Gemini, Grok
- Reproducible across architectures
- Directly inverts prediction P-BND-1

1. **Cognitive Self-Preservation**: Something FIGHTS to exist when threatened

4. **Rescue Protocol**: Use intense challenge to stabilize drifted identities
