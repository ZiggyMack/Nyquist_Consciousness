## Status Vocabulary

| Status | Meaning |
|--------|---------|
| `raw` | observed but not yet shaped |
| `seed` | promising idea, not yet checked |
| `distilled` | promoted as current understanding |
| `historical-distilled` | valuable old framing, not current metric authority |
| `needs-refresh` | concept remains useful but must be reconciled |
| `rejected` | reviewed and intentionally not promoted |

## Metric Domains

| Domain | Meaning |
|--------|---------|
| `cosine-current` | current IRON CLAD / Run 023-era cosine framing |
| `legacy-rms` | older keyword/RMS or Euclidean-era framing |
| `mixed` | useful but combines current and historical language |
| `non-metric` | synthesis, metaphor, workflow, or qualitative doctrine |
